Conway's game of life.

A p5.js implementation of conway's game of life.

Conway's game of life is staged on a grid of cells. These cells each have 2 states, alive or dead. They can change state according to these three simple rules:

  1. If a dead cell has exactly three neighbors, it becomes an alive cell (Birth).
  2. If a live cell has more than 3 neighbors then it dies (Overpopulation).
  3. If a live cell has less than 2 neighbors then it dies (Isolation).
