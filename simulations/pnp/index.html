Predator and Prey

A p5.js implementation of Predator and Prey.

This strange version of predator and prey functions according to some simple rules:

  • If a predator cell has any predator neighbors, it dies.
  • If a prey cell has any predators as neighbors, it dies.
  • If there are no prey around a predator, it dies.
  • If a dead cell has 3 prey neighbors, it becomes a prey cell.
  • If a dead cell has any predator neighbors, the chance of it becoming a predator cell is equal to the ((the number of predators) - 1)/(the number of predators)
